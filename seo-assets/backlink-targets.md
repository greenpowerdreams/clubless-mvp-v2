# Backlink Targets — 30 High-Value Sites

**Goal:** Build 25+ referring domains in the first 90 days. Priority on local + nightlife + indie startup sites because they pass more topical relevance for "clubless" and "clubless collective" than generic directories.

**How to use:** Work from the top down. Send 5 outreach emails per day. Track status in a spreadsheet (URL / contact / status / date).

## Link Anchor Text Goals

When negotiating press / sponsored placements, try to get these anchor texts (in priority order):

1. **Clubless** (hardest to get but most valuable)
2. **Clubless Collective**
3. **clublesscollective.com**
4. Branded + keyword: "Clubless Collective, Seattle's event platform"

Avoid: "click here", "this website", "read more" — wasted anchor text.
